# Accessibility Resources

- Complete List of Web Accessibility Evaluation Tools
  http://www.w3.org/WAI/ER/tools/complete
- W3C Markup Validation Service: Checks Web documents in formats like HTML and
  XHTML for conformance to W3C Recommendations and other standards
  http://validator.w3.org/
- Access Color – Online Tool for Color Contrast
  http://www.accesskeys.org/tools/color-contrast.html
- SiteCheck: Check, find and fix broken links, spelling mistakes and accessibility errors
  http://siteimprove.com/Solutions/SiteCheck.aspx
- Webagogo: Free online tool to test the quality of your Website, for accessibility, usability,
  etc.
  http://www.Webagogo.be/
- Web Accessibility Inspector: Will determine whether your Website can easily be viewed
  by the elderly and those with visual impairments
  http://www.fujitsu.com/global/accessibility/assistance/wi/
